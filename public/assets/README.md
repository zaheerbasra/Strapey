# Strapey Brand Assets

This directory contains all brand assets for the Strapey brand identity system.

## Brand Identity

**Brand Name:** Strapey  
**Motto:** Crafted for Wild  
**Industry:** Premium Hunting & Kitchen Knives

## Asset Files

### Logo Files

#### `logo.svg` (300×120px)
- Primary vertical logo
- Forest green background
- Knife icon with mountain silhouette
- Brand name and motto
- **Use:** Website headers, email signatures, print materials

#### `logo-horizontal.svg` (400×80px)
- Horizontal layout for constrained spaces
- Charcoal background
- Compact knife icon with divider
- **Use:** Navigation bars, product packaging

#### `logo-icon.svg` (100×100px)
- Circular icon with knife illustration
- Forest green background with "S" initial
- **Use:** Social media profiles, app icons

#### `favicon.svg` (64×64px)
- Simplified browser icon
- **Use:** Browser tabs, bookmarks

#### `banner-hero.svg` (1200×400px)
- Large hero banner with Damascus pattern
- Mountain silhouette and full knife illustration
- **Use:** Website hero sections, large promotional banners

### Marketing Assets

#### `ebay-store-banner.svg` (1200×270px)
- eBay store header design
- Category tags included
- **Use:** eBay Store settings upload

#### `social-media-post-template.svg` (1080×1080px)
- Square format for Instagram/Facebook
- Product highlight area with CTA
- **Use:** Social media product showcases

#### `facebook-cover-banner.svg` (1200×628px)
- Facebook cover photo format
- Large knife illustration with brand elements
- **Use:** Facebook page cover, LinkedIn banner

#### `website-hero-section.svg` (800×600px)
- Homepage hero banner
- "EXPLORE COLLECTION" CTA
- **Use:** Website homepage hero section

### Style Assets

#### `brand-colors.css`
- CSS variables for all brand colors
- Example usage and component styles
- **Use:** Import into website stylesheets

## Color Palette

### Primary Colors
- **Forest Green:** `#1a4d2e`
- **Charcoal Black:** `#2c2c2c`
- **Forged Steel Gray:** `#6b7280`

### Accent Colors
- **Burnt Orange:** `#d97706`
- **Bronze:** `#92400e`
- **Copper:** `#b45309`

## Typography

**Headings:** 'Bebas Neue', 'Impact', 'Arial Black', sans-serif  
**Body Text:** 'Open Sans', 'Helvetica Neue', Arial, sans-serif

## Export Formats

All SVG files can be exported to:
- **SVG** - Scalable vector (preferred for web)
- **PNG** - Transparent background (web/digital)
- **JPG** - Solid background (print)

## Usage Guidelines

1. Always maintain logo proportions
2. Use on contrasting backgrounds
3. Apply brand colors consistently
4. Follow typography guidelines
5. Reference `BRAND_GUIDE.md` for complete guidelines
